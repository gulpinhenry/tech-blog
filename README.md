## Installation

1. [Install Node.js](https://nodejs.org/en/download/)
2. [Install MySQL](https://www.mysql.com)

3. Create database
   
        mysql -u root -p
        source ./db/schema.sql

4. Install JavaScript packages given in package.json

        npm install


5. Seed database:

        npm run seed

## Usage

        npm start
